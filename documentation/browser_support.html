We know that widgets will not work on these browsers:

Internet Explorer 8 and 9
currently widgets need to to cross domain call to Sabre api. These HTTP requests must have custom headers with auth token, while IE 8 and 9 (before IE 10) does not support sending CORS requests (preflight OPTIONS request is needed, it is not supported). Also plain JSONP will not work either as POST call is needed. Details on IE 8 and 9 implementation

Widgets were not tested on:

Opera
because first of small market share, second Opera starting with version 15 uses Chromium, and we are testing on Chrome.
